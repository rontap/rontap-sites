about tapcalk 6

open_in_new
Unlike previous tapcalk versions, this iteration has multiple parts under construction
It has been decided that a complete rewrite would be too time consuming, so a modularisation and modernisation plan was layed out. The parts are:

- NXT : Plans to integrate into the nex RTS athmosphere, mostly by UI and JS functions rewrites and rearrangements
- DESIGN : uses the NXT design to change the appearance and redesign the document and the parsing methods.
- EXPRESS : aims to modularise tapcalk into way smaller packages, and achieving the dream of a usable API, called TapCalkExpress
- EEE : full translation to English
- TOOLS : function parsing and not strictly calculation related updates

Down below you can see the current status of the updates
